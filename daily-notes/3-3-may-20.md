## Exercise: l33t

Write a function `l33t(str)` that takes a string and replaces vowels with their
l33t counterparts: `a = @, e = 3, i = 1, o = 0, u = |_|`

Sample usage:

```
    leet('here you are'); // returns h3r3 y0|_| @r3
```
